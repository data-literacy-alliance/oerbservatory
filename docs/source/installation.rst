Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/oerbservatory>`_ with uv:

.. code-block:: console

    $ uv pip install oerbservatory

or with pip:

.. code-block:: console

    $ python3 -m pip install oerbservatory

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/oerbservatory.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/oerbservatory.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/oerbservatory.git
    $ cd oerbservatory
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
